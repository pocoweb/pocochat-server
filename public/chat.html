About this project

This project is a simple javascript based multi-user chat which utilizes several JavaScript APIs to maximize potential and minimize effort.

The Parse API provides a simple, secure and efficient backend solution to hold all of the data necessary to make this application such as usernames, passwords and the messages being sent by users.

The jQuery API allows us to provide the user with a clean and easy to use interface while writing a considerable less amount of code than by using native libraries.

The text editor used to produce and manage all project files is Sublime Text 3.
